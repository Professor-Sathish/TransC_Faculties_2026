# The system provides the following functionalities:

# Account Creation:
Allows a user to create a new bank account by entering an account number, PIN, and initial balance. The account is activated and unlocked by default.
 
# Deposit:
Enables money to be deposited into an existing account after validating the account number.

# Withdrawal:
Allows withdrawal of money after verifying the correct PIN and ensuring sufficient balance.

# Balance Enquiry:
Displays the current balance of an account after successful PIN verification.

# PIN Change:
Allows the account holder to change the PIN after entering the correct existing PIN.

# Money Transfer:
Transfers funds from one account to another, ensuring both accounts are valid and the sender has sufficient balance.

# Wrong PIN Handling and Account Locking:
If an incorrect PIN is entered three times consecutively, the account is automatically locked to prevent unauthorized access.

# Unlock Account:
Resets the lock status and wrong PIN count, allowing the account to be used again.

# Delete Account:
Deactivates an account so that it cannot be accessed further.
